# Project Title Java program

This project was to learn java and create a program that could run using variables, conditions, loops, and other items.

## Instructions for Build and Use

Steps to build and/or run the software:

1. First open Visual Studio Code and make a .java file
2. Install the red hat extension for helping with java
3. Make a simple program with the main class and another class
4. Initialize variables, build functions in classes, implement them in the main program
5. Realize that red hat is not sufficient to run java and find a JDK from Microsoft to debug and run it

Instructions for using the software:

1. Open up the Main.java file
2. Run the program
3. Let the IDE do its thing

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Download the JDK from this website: https://learn.microsoft.com/en-us/java/openjdk/download
* Install the red hat extension (acts as a JRE)
